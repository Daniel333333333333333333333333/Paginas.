¿Que es css?

Se trata de un «idioma» o lenguaje, como podría ser el inglés o el alemán, que los navegadores web como Chrome o Firefox conocen y pueden entender, nuestro objetivo como diseñadores y programadores web es precisamente ese: aprender el idioma e indicarle al navegador lo que debe hacer.

CSS

Significado de las siglas css

Las siglas CSS (Cascading Style Sheets) significan Hojas de estilo en cascada.

Para que sirve css

CSS sirve para poder dar un estilo a cualquier web y hacerlo con las reglas que se deseen.

Componentes de una hoja de estilo en CSS

Una hoja de estilo se compone de un listado de reglas o comandos que constan de un selector y una declaración.
